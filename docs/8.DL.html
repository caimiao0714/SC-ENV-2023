autoML框架

  • AutoGluon for tabular prediction
    • AutoGluon documentation: AutoGluon enables easy-to-use and easy-to-extend AutoML with a focus on automated stack ensembling, deep learning, and real-world applications spanning text, image, and tabular data. Intended for both ML beginners and experts, AutoGluon enables you to:
    • Quickly prototype deep learning and classical ML solutions for your raw data with a few lines of code.
    • Automatically utilize state-of-the-art techniques (where appropriate) without expert knowledge.
    • Leverage automatic hyperparameter tuning, model selection/ensembling, architecture search, and data processing.
    • Easily improve/tune your bespoke models and data pipelines, or customize AutoGluon for your use-case.
  • H2O4GPU: The Python API builds upon the easy-to-use scikit-learn API and its well-tested CPU-based algorithms. It can be used as a drop-in replacement for scikit-learn (i.e. import h2o4gpu as sklearn) with support for GPUs on selected (and ever-growing) algorithms.
  • Auto-PyTorch: While early AutoML frameworks focused on optimizing traditional ML pipelines and their hyperparameters, another trend in AutoML is to focus on neural architecture search. To bring the best of these two worlds together, we developed Auto-PyTorch, which jointly and robustly optimizes the network architecture and the training hyperparameters to enable fully automated deep learning (AutoDL).

安装包

以下展示的是Linux系统上使用pip安装基于GPU的AutoGLuon。

python3 -m pip install -U pip
python3 -m pip install -U setuptools wheel

# Here we assume CUDA 10.1 is installed.  You should change the number
# according to your own CUDA version (e.g. mxnet_cu100 for CUDA 10.0).
python3 -m pip install -U "mxnet_cu101<2.0.0"
python3 -m pip install autogluon

拟合模型和结果预测

from autogluon.tabular import TabularPredictor
predictor = TabularPredictor(label=COLUMN_NAME).fit(train_data=TRAIN_DATA.csv)
predictions = predictor.predict(TEST_DATA.csv)
